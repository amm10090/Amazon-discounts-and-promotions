简介

这是一个专注于亚马逊折扣商品的查询平台，提供以下功能：

  • 实时获取亚马逊折扣商品信息
  • 查看详细的商品信息
  • 多语言支持

折扣商品

GET /api/v1/products/deals

获取商品详情

GET /api/v1/products/:asin

路径参数

参数名 类型 描述
asin string 商品的ASIN编号

错误代码

状态码 错误代码 描述
400 INVALID_REQUEST 请求参数无效
401 UNAUTHORIZED 未授权访问
404 NOT_FOUND 资源未找到
429 RATE_LIMIT_EXCEEDED 超出请求限制
500 INTERNAL_ERROR 服务器内部错误

限流说明

为了确保服务质量，我们对API访问进行了限制：

  • 每个IP地址每分钟最多可发起30次请求
  • 超出限制将返回429状态码
