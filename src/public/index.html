简介

这是一个整合了亚马逊商品数据的API平台，提供以下功能：

  • 搜索商品并获取折扣信息
  • 获取详细的商品信息
  • 查询商品变体
  • 多语言支持

认证

所有API请求都需要在Header中包含以下认证信息：

{
  "Authorization": "Bearer YOUR_API_KEY"
}
请联系管理员获取API密钥。

API端点

搜索商品

GET /api/v1/products/search

请求参数

参数名 类型 必填 描述
keywords string 是 搜索关键词
searchIndex string 否 商品分类(默认: All)
itemCount number 否 返回结果数量(默认: 10)

示例请求

curl -X GET "https://api.example.com/api/v1/products/search?keywords=iphone&searchIndex=Electronics&itemCount=5" \
-H "Authorization: Bearer YOUR_API_KEY"

示例响应

{
  "status": "success",
  "message": "搜索成功",
  "data": {
    "products": [
      {
        "asin": "B08PP5MSVB",
        "title": "Apple iPhone 12, 64GB, Black - Unlocked",
        "image": "https://m.media-amazon.com/images/I/31hH5lo4O+L._SL75_.jpg",
        "price": "$599.99",
        "url": "https://www.amazon.com/dp/B08PP5MSVB"
      }
    ],
    "total": 1
  }
}

获取商品详情

GET /api/v1/products/:asin

路径参数

参数名 类型 描述
asin string 商品的ASIN编号

示例请求

curl -X GET "https://api.example.com/api/v1/products/B08PP5MSVB" \
-H "Authorization: Bearer YOUR_API_KEY"

获取商品变体

GET /api/v1/products/:asin/variations

路径参数

参数名 类型 描述
asin string 商品的ASIN编号

示例请求

curl -X GET "https://api.example.com/api/v1/products/B08PP5MSVB/variations" \
-H "Authorization: Bearer YOUR_API_KEY"

错误代码

状态码 错误代码 描述
400 INVALID_REQUEST 请求参数无效
401 UNAUTHORIZED 未授权访问
404 NOT_FOUND 资源未找到
429 RATE_LIMIT_EXCEEDED 超出请求限制
500 INTERNAL_ERROR 服务器内部错误

限流说明

为了确保服务质量，我们对API访问进行了限制：

  • 每个API密钥每分钟最多可发起60次请求
  • 每个IP地址每分钟最多可发起30次请求
  • 超出限制将返回429状态码
